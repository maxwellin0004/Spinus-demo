# Workflow Short Video Script Creator

Use this skill for fast short-video script generation.

## Best-Fit Tasks

- 30s to 90s scripts
- hooks and openings
- dialogue-first short scenes
- imitation drafts based on a reference structure
- prompt-ready script drafts for later video generation

## Output Preference

Prefer:

- scene timing
- visual/action cues
- spoken lines
- clear closing line or repeated verdict when relevant
