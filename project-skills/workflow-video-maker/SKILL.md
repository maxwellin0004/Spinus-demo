# Workflow Video Maker

Use this skill after script creation, when the output must become an implementation plan.

## Primary Scope

- map script sections to compositions or scenes
- connect assets, subtitles, voiceover, and timing to repository files
- prepare a Remotion-ready execution path

## Repository Mapping

- `data/templates/`
- `video-app/src/compositions/`
- `video-app/src/scenes/`
- `video-app/src/data/`
- `video-app/public/`
