## 阶段2：模板判断与抽象

| 问题 | 回答 |
|---|---|
| 这条视频属于什么模板类型？ | 横屏 cinematic knowledge essay，中文长口播职场/认知解释纪录片。 |
| 为什么不适合归入现有模板？ | 现有模板多为交易解释、AI/HUD、漫画、心理极简、档案页或文学蒙太奇；参考片核心是“全屏真实感 B-roll + 中英硬字幕 + 章节概念大字 + 长口播论证”，页面系统和镜头语法都不同。 |
| 是否必须新建模板？ | 是。否则会把资料片解释误套成现有 HUD/卡片/漫画结构。 |
| 是否必须新建 scene 系统？ | 是。需要 B-roll 母版、章节冲击字、案例蒙太奇、机制解释和结尾收束等新 scene。 |
| video-app 中哪些只能复用基础能力？ | `SubtitleTrack`、`AudioTrackLayer`、`audioTypes`、字体 token、Remotion 基础动画、`staticFile` 图片加载。 |
| 核心新风格语法 | 暖色/冷灰电影资料片、轻胶片颗粒、全屏图像、低 UI 密度、概念词压画面。 |
| 核心新页面系统 | 无卡片的 full-bleed documentary B-roll 页面系统。 |
| 核心新动画/转场语法 | 3-7 秒硬切 + Ken Burns 慢推/慢横移 + 章节词短促冲击。 |
| 是否需要外部素材？ | 最佳成片需要更多办公室/会议/白板/演讲/书架视频素材。当前预览优先使用本地 public 素材，不足部分用静态图慢推替代。 |

|---|---|
| 模板名称候选 | `cognitive-documentary-essay`、`cinematic-knowledge-essay`、`bilingual-broll-explainer` |

| 适合方向 | 职场沟通、结构化表达、学习方法、管理认知、产品/组织效率 |
| 不适合方向 | 强操作教程、纯数据看盘、游戏解说、漫画剧情、需要大量 UI 步骤演示的工具教程 |
