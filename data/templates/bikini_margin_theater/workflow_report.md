### 2. 标题包装拆解
| 维度 | 观察 |
|---|---|
| 形式 | 黑底 + 白色毛笔字（大字号） |
| 首屏文案 | 0-5 秒分段写出“这就是…比奇堡最隐秘的财富法则”（部分字样以抽样可见） |
| 目标 | 在 2 秒内建立“强观点+故事感” |
| 规则 | 先黑底高反差，再切入剧情舞台 |

### 3. 简介拆解
| 项目 | 结果 |
|---|---|
| 视频内可见简介文案 | 不确定（视频画面未直接展示平台简介区） |
| 推测策略 | 与标题同主题：故事化经济学解释 |
| 置信度 | 低（无平台页面证据） |

### 4. Hook 拆解
| 时间 | 画面 | 动效 | 字幕/音频配合 |
|---|---|---|---|
| 0.00-1.50s | 黑底，第一段毛笔字出现 | 逐字/逐句写入，轻微抖动感 | 无角色，口播直接抛观点 |
| 1.50-3.50s | 黑底继续补全标题 | 续写式 reveal | 字幕短句，语速偏快 |
| 3.50-5.20s | 黑底完成核心标题 | 完成后短停顿 | 准备切主场景 |
| 5.20-6.00s | 切到海底固定舞台 | 硬切/闪切到场景 | 口播进入故事设定 |

### 5. 内容结构拆解
| 段落 | 时间（约） | 作用 | 主要呈现 |
|---|---|---|---|
| A Hook | 0-6s | 抓注意力 | 黑底毛笔字 |
| B 价格设定 | 6-28s | 建立“标价/优惠”问题 | 白板+汉堡价格+角色入场 |
| C 边际逻辑 | 28-65s | 引出边际成本/收益 | 公式化字幕+蟹老板/海绵宝宝道具 |
| D 选择后果 | 65-110s | 用户侧决策后果 | 派大星/章鱼哥气泡思考 |
| E 方法升级 | 110-146s | 从消费转到时间/注意力投资 | 多角色轮流讲解+关键词 |
| F 收束 | 146-169s | 夜景情绪落点 | 夜景背景+毛笔总结 |

### 6. 文稿风格拆解
| 维度 | 观察 |
|---|---|
| 句式 | 短句+结论句，常见“不是A而是B” |
| 语气 | 讲故事式解释，不是论文式解释 |
| 信息密度 | 中高（每 2-4 秒给一个新信息点） |
| 修辞 | 角色拟人、反问、对比 |

### 7. 画面结构拆解
| 维度 | 观察 |
|---|---|
| 舞台策略 | 长时间固定在同一海底街景（单舞台） |
| 主体层级 | 背景层 > 角色层 > 道具层（白板/价格牌/气泡）> 字幕层 |
| 页面母版 | `黑底标题母版` / `固定舞台演算母版` / `气泡对话母版` / `夜景收束母版` |
| 角色调度 | 主要通过左右中位置切换和进出场实现节奏 |

### 8. 画面节奏拆解
| 指标 | 观察 |
|---|---|
| 硬切频率 | 低（少量硬切，主要在 Hook->舞台、日景->夜景） |
| 同舞台内变化 | 高（道具/角色 2-4 秒发生一次局部变化） |
| 观感节奏 | “镜头稳定 + 画面元素高频替换” |

### 9. 动画与特效拆解
| 类型 | 作用 |
|---|---|
| 文字/毛笔 reveal | 开场冲击 |
| 角色 pop-in | 强化段落转折 |
| 气泡弹出 | 表达心理/观点 |
| 道具叠加（货币/价格牌） | 把抽象概念具象化 |
| 轻缩放/轻位移 | 防止静态画面疲劳 |

### 10. 转场拆解
| 时间点 | 转场方式 | 备注 |
|---|---|---|
| 约 5-6s | 黑底->舞台硬切 | 典型开场转叙事 |
| 约 145-150s | 日景->夜景硬切+色调切换 | 进入总结段 |
| 其余多数段 | 伪转场（同场景换角色/换道具） | 非传统镜头切换 |

### 11. 字幕拆解
| 维度 | 观察 |
|---|---|
| 位置 | 底部居中 |
| 样式 | 白字 + 黑描边，易读性优先 |
| 分句 | 短句分块，约 1-3 秒一条 |
| 与口播关系 | 基本同步口播，不是独立文案 |

### 12. 音频拆解
| 维度 | 观察 |
|---|---|
| 主轨 | 单人中文口播 |
| 节奏 | 中快语速，停顿短 |
| BGM | 存在感低到中（不抢口播） |
| SFX | 可能存在轻提示（不确定，未做频谱拆解） |

### 13. 素材类型拆解
| 素材类型 | 用法 |
|---|---|
| 动画IP背景 | 固定舞台底图 |
| 角色图 | 观点承载体 |
| 白板/价格牌/气泡 | 信息结构化道具 |
| 毛笔字 | 章节强记忆锚点 |

### 14. 固定规则与变量
| 固定项 | 变量项 |
|---|---|
| 0-5 秒黑底毛笔 Hook | Hook 文案主题 |
| 单舞台为主，不靠频繁切镜 | 角色数量、站位、出场顺序 |
| 道具层叠解释抽象概念 | 白板内容、公式和数字 |
| 底部白字黑描边字幕 | 每条字幕文本与时长 |
| 结尾夜景+大字收束 | 收尾口号与 CTA |

### 15. 最值得复用的模板规律
| 规律 | 可复用规则化表达 |
|---|---|
| 强 Hook | `黑底大字 + 单观点 + 5秒内完成` |
| 稳镜头高信息 | `固定舞台 + 高频道具替换` |
| 抽象具象化 | `每个概念绑定一个角色或道具` |
| 收尾记忆点 | `夜景换色 + 口号大字 + 1秒停留` |

## 阶段2：模板判断与抽象

### 先答9问
| 问题 | 结论 |
|---|---|
| 1. 属于什么模板类型 | 角色演绎型“固定舞台叙事演算模板” |
| 2. 为什么不适合归入现有模板 | 现有模板偏卡片/HUD/资讯页，不具备“单舞台+气泡对话+毛笔收束”语法 |
| 3. 是否需要新建模板 | 需要 |
| 4. 是否需要新建 scene 系统 | 需要 |
| 5. 哪些只能复用基础能力 | 字幕组件、音频轨组件、Remotion 基础动画函数 |
| 6. 核心新风格语法 | 黑底毛笔钩子 + 海底舞台 + 角色冲突演绎 |
| 7. 核心新页面系统 | `Hook页 -> 演算页 -> 辩论页 -> 方案页 -> 夜景收束页` |
| 8. 核心新动画/转场语法 | 同场景内对象位移/缩放/弹入，少硬切 |
| 9. 是否需要外部素材 | 可选；本次优先使用本地抽帧和程序化图形，音频用 ElevenLabs |

| 项目 | 结果 |
|---|---|
| 模板名候选 | `bikini_margin_theater` / `stage_debate_ledger` |
| 适用赛道 | 泛知识、认知升级、效率方法 |
| 适用平台 | B站、视频号、YouTube |
| 视频类型 | 横屏叙事解释视频 |
| 推荐时长 | 60-95 秒 |
| 目标受众 | 想快速理解抽象概念的职场/学习用户 |
| 转化目标 | 评论互动、收藏、关注系列化内容 |
